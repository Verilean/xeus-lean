xeus-lean

A Lean 4 Jupyter kernel that runs entirely in the browser (WebAssembly), with practical, application-oriented tutorials.

Documentation

  • Learn Lean 4 — a practical, application-developer tutorial: values, data structures, IO, files, processes, networking, concurrency, JSON, macros, coming from Haskell, and why the reference-counted runtime behaves like C.
  • JupyterLite — run every chapter live in your browser (the xlean WASM kernel, no install required).

Visual mathematics

  • Visual Complex Analysis — an interactive, notebook-based walk-through, runnable in the browser.
  • Visual Real Analysis — likewise, for real analysis.
  • Visual Geometric Algebra — Clifford algebra in the Visual Complex Analysis spirit: bivectors, rotors, reflections from pictures.
